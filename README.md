# Netflix Dataset – Excel Data Cleaning Project

## Overview
This project involves cleaning and preparing the Netflix Movies and TV Shows dataset using Microsoft Excel. The objective is to enhance data quality, consistency, and readiness for analysis.

## Dataset
- *Name:* Netflix Movies and TV Shows Dataset  
- *Format:* CSV

## Tools Used
- Microsoft Excel

## Data Cleaning Steps
The following actions were performed during the data cleaning process:

- Imported the CSV file into Excel
- Created separate Raw and Working datasets to preserve the original data
- Identified missing values using filters and conditional formatting
- Checked for and handled duplicate records
- Standardized text fields using TRIM and PROPER functions
- Converted date fields into a consistent yyyy-mm-dd format
- Generated a final cleaned dataset
- Documented data quality observations

## Files Included
- *Raw_Data.xlsx* – Original dataset
- *Cleaned_dataset.xlsx* – Cleaned and formatted dataset
- *cleaned_dataset.csv* – Final cleaned dataset in CSV format

## Notes
Some records contain missing values in non-mandatory fields such as director. These values were retained as they do not significantly affect the overall dataset quality.
